## WSTG-CONF-05 — تعداد واجهات الإدارة للبنية التحتية والتطبيق (Enumerate Infrastructure and Application Admin Interfaces)

تمثل واجهات الإدارة (Administrative interfaces) مستوى التحكم بالإدارة للتطبيق وللبنية التحتية الأساسية الخاصة به، وغالباً ما تمنح وصولاً بامتيازات (Privileged access) إلى إعدادات النظام، وبيانات المستخدمين، والعمليات البرمجية في جانب الخادم (Server-side). يعطي المهاجمون الأولوية لتعداد هذه الواجهات من خلال التخمين العشوائي للمجلدات (Directory Brute-forcing)، أو تحليل ملف `robots.txt` ، أو مراقبة أنماط عناوين URL المتوقعة للعثور على نقاط دخول قد تفتقر إلى المصادقة (Authentication) القوية أو تعتمد على بيانات الاعتماد الافتراضية (Default Credentials). إن اكتشاف لوحة إدارة مكشوفة يزيد بشكل كبير من خطر الاختراق الكامل للنظام، أو تعديل البيانات غير المصرح به، أو تعطل الخدمة. غالباً ما توجد هذه الواجهات على منافذ غير قياسية (Non-standard ports) أو نطاقات فرعية (Subdomains)، مما يجعلها أهدافاً رئيسية للماسحات الآلية (Automated scanners) والاستطلاع اليدوي (Manual reconnaissance).


| الحقل | القيمة |
|---|---|
| **WSTG ID** | WSTG-CONF-05 |
| **CWE** | CWE-425 |
| **حالة الاختبار** | لم يتم تنفيذه |
| **الخطورة** | متوسطة / عالية* |

> *تصبح الخطورة عالية إذا كانت الواجهة تسمح بتغييرات في إعدادات النظام بالكامل، أو إدارة المستخدمين، أو تسريب البيانات (Data exfiltration) دون وجود المصادقة متعددة العوامل (Multi-Factor Authentication - MFA).

### هل يمكن اكتشاف واجهات الإدارة عبر التخمين (Directory Fuzzing) أو الاستطلاع؟
- [ ] لا — لا توجد واجهات إدارية مكشوفة أو يمكن اكتشافها  
- [ ] نعم — الواجهات قابلة للاكتشاف ولكن الوصول **مقيد** لنطاقات عناوين IP الداخلية  
- [ ] نعم — الواجهات **قابلة** للاكتشاف ومتاحة للعامة  

### هل يتم فرض المصادقة على بوابات الإدارة المكتشفة؟
- [ ] نعم — يتم فرض المصادقة متعددة العوامل (MFA)  
- [ ] نعم — يتم فرض المصادقة أحادية العامل (Single-factor authentication)  
- [ ] لا — لا توجد مصادقة مطلوبة للوصول إلى الواجهة *(حرِج)*  

### هل مسارات الإدارة مخفية أو غير قياسية لمنع اكتشافها؟
- [ ] نعم — تستخدم المسارات تسميات غير قياسية، أو عشوائية، أو غير متوقعة  
- [ ] لا — تستخدم المسارات اتفاقيات التسمية الشائعة (مثل `/admin` أو `/manager` أو `/console`) و**يمكن** تخمينها بسهولة  

### هل توفر الواجهة وصولاً إلى وظائف النظام أو التطبيقات الحساسة؟
- [ ] لا — الواجهة هي لوحة تحكم للقراءة فقط (Read-only dashboard) مع تأثير ضئيل  
- [ ] نعم — الواجهة **يمكنها** تعديل إعدادات التطبيق أو بيانات المستخدم  
- [ ] نعم — الواجهة **يمكنها** تنفيذ أوامر على مستوى النظام أو القيام بإدارة قواعد البيانات
