## WSTG-ATHZ-04 — اختبار مراجع الكائنات المباشرة غير الآمنة (Insecure Direct Object References)

تحدث مراجع الكائنات المباشرة غير الآمنة (Insecure Direct Object References - IDOR) عندما يوفر التطبيق وصولاً مباشراً إلى الكائنات بناءً على مدخلات مقدمة من المستخدم دون إجراء فحص للصلاحيات (Authorization) للتحقق من أذونات مقدم الطلب. تؤثر هذه الثغرة بشكل كبير على السرية (Confidentiality) والنزاهة (Integrity)، حيث تتيح للمهاجمين عرض أو تعديل أو حذف البيانات التابعة لمستخدمين آخرين أو للنظام. تظهر عادةً في معلمات URL، أو بيانات جسم طلب POST، أو مفاتيح JSON التي تشير إلى مفاتيح قاعدة بيانات داخلية، أو أسماء ملفات، أو معرفات حسابات. من منظور المهاجم، يتضمن الاستغلال (Exploit) التلاعب بهذه المعرفات — غالباً من خلال زيادة الأرقام الصحيحة أو التخمين (Fuzzing) لمعرفات UUID — للوصول إلى موارد خارج نطاق صلاحياتهم المصرح بها.

### هل يستخدم التطبيق معرفات كائنات مباشرة في معلمات الطلب؟
- [ ] لا — يستخدم التطبيق مراجع غير مباشرة أو خرائط مشفرة *(الأكثر أماناً)*  
- [ ] نعم — يستخدم التطبيق معرفات غير قابلة للتنبؤ (مثل معرفات UUID الطويلة أو الهاشات Hashes)  
- [ ] نعم — يستخدم التطبيق معرفات قابلة للتنبؤ (مثل الأرقام المتسلسلة أو أسماء المستخدمين)  

### هل يتم إجراء فحص للصلاحيات من جانب الخادم (Server-side) لكل طلب يتضمن معرف كائن؟
- [ ] نعم — **لا يمكن** تجاوز فحوصات جانب الخادم لأي معرف  
- [ ] نعم — توجد فحوصات من جانب الخادم ولكن **يمكن التجاوز** عبر تلوث المعلمات (Parameter Pollution) أو التلاعب بالترويسات (Header Manipulation)  
- [ ] لا — **لا يتم تطبيق** أي فحص للصلاحيات للتحقق من ملكية الكائن المطلوب  

### هل يمكن للمهاجم الوصول إلى كائنات تابعة لمستخدمين آخرين أو تعديلها (IDOR أفقي - Horizontal IDOR)؟
- [ ] لا — الوصول إلى بيانات المستخدمين الآخرين **غير ممكن**  
- [ ] نعم — عرض بيانات المستخدمين الآخرين **ممكن** ولكن التعديل **غير ممكن**  
- [ ] نعم — كل من العرض والتعديل لبيانات المستخدمين الآخرين **ممكن** *(حرجة - Critical)*  

### هل من الممكن الوصول إلى كائنات على المستوى الإداري أو مستوى النظام (IDOR رأسي - Vertical IDOR)؟
- [ ] لا — الوصول إلى الكائنات ذات الصلاحيات العالية **غير ممكن**  
- [ ] نعم — الوصول إلى الإعدادات الإدارية أو ملفات النظام **ممكن**  

### هل يقوم التطبيق بتسريب معرفات الكائنات عبر البحث أو البيانات الوصفية (Metadata) أو نقاط النهاية الأخرى؟
- [ ] لا — المعرفات **لا يتم كشفها** بشكل غير مقصود  
- [ ] نعم — **يمكن** حصر (Enumerate) معرفات المستخدمين/الكائنات الآخرين من خلال نقاط نهاية ثانوية أو ملفات تعريفية عامة
