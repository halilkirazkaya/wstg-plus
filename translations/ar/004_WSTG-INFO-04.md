## WSTG-INFO-04 — حصر التطبيقات على خادم الويب (Enumerate Applications on Webserver)

حصر التطبيقات (Application enumeration) هو عملية تحديد جميع تطبيقات وخدمات الويب المستضافة على خادم ويب (web server) واحد أو عنوان IP معين. نظرًا لأن خوادم الويب الحديثة غالبًا ما تستضيف عدة مضيفات افتراضية (virtual hosts)، أو بيئات اختبار (staging environments) قديمة، أو واجهات إدارية على منافذ ومسارات غير قياسية، فإن الفشل في تحديد هذه الأصول المخفية يترك جزءًا كبيرًا من سطح الهجوم (attack surface) دون اختبار. يستخدم المهاجمون تقنيات مثل نقل نطاق نظام أسماء النطاقات (DNS zone transfers)، والتخمين العشوائي للمضيفات الافتراضية (virtual host brute-forcing)، والبحث العكسي عن عناوين البروتوكول (reverse IP lookups) لاكتشاف نقاط الدخول المنسية أو الغامضة، والتي غالبًا ما تكون أقل أمانًا من تطبيق الإنتاج الأساسي.

### هل هناك عناوين IP متعددة أو أسماء مستعارة لنظام أسماء النطاقات (DNS aliases) مرتبطة بالبنية التحتية المستهدفة؟
- [ ] لا — يوجد عنوان IP واحد وسجل A-record واحد فقط.
- [ ] نعم — تم تحديد عناوين IP متعددة أو أسماء مستعارة، مما يؤدي إلى توسيع نطاق الفحص.

### هل يستضيف خادم الويب عدة مضيفات افتراضية (vhosts) على نفس عنوان IP؟
- [ ] لا — النطاق الأساسي فقط هو ما يخدمه الخادم.
- [ ] نعم — تم تحديد مضيفات افتراضية إضافية ولكن الوصول إليها غير ممكن (مثال: 403 Forbidden).
- [ ] نعم — تم تحديد مضيفات افتراضية مخفية أو خاصة بالتطوير أو الاختبار، وهي متاحة للوصول.

### هل يتم استضافة تطبيقات على منافذ غير قياسية (non-standard ports)؟
- [ ] لا — المنافذ القياسية فقط (80/443) هي المفتوحة.
- [ ] نعم — توجد منافذ غير قياسية مفتوحة ولكنها لا تستضيف تطبيقات ويب.
- [ ] نعم — تم تحديد تطبيقات ويب إضافية على منافذ غير قياسية (مثال: 8080، 8443، 9000).

### هل يتم تعيين تطبيقات متميزة لمسارات URI محددة؟
- [ ] لا — تطبيق واحد يخدم دليل الجذر بالكامل.
- [ ] نعم — تم اكتشاف تطبيقات متعددة من خلال حصر الأدلة (Directory enumeration) (مثال: `/api`, `/portal`, `/backup`).

### هل تكشف خدمات الاستطلاع الخارجية عن تطبيقات تاريخية أو ثانوية؟
- [ ] لا — لا توجد نتائج مهمة من Shodan أو Censys أو Wayback Machine.
- [ ] نعم — تكشف اللقطات التاريخية أو عمليات فحص الخدمات عن تطبيقات نشطة حاليًا ولكنها غير مرتبطة بشكل مباشر.
