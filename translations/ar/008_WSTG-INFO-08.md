## WSTG-INFO-08 — تحديد بصمة إطار عمل تطبيقات الويب (Fingerprint Web Application Framework)

تتضمن عملية تحديد بصمة إطار عمل تطبيقات الويب (Fingerprinting) تحديد مكدس التكنولوجيا (Technology Stack) وإصدارات البرامج المحددة المستخدمة في بناء وتشغيل التطبيق. يقوم المهاجمون بتحليل ترويسات استجابة (HTTP response headers)، وملفات تعريف الارتباط (Cookies) الخاصة بإطار العمل، وهياكل الملفات المتوقعة، وعناصر JavaScript الفريدة لتحديد ما إذا كان الهدف يشغل منصات مثل React أو Angular أو Django أو Spring. تعد مرحلة الاستطلاع (Reconnaissance) هذه حيوية لأنها تسمح للمختبر بتضييق نطاق سطح الهجوم (Attack Surface) ليشمل الثغرات المعروفة (CVEs) ونقاط ضعف التكوين الشائعة الخاصة بهذا الإطار المحدّد. من خلال فهم الهندسة المعمارية الأساسية، يمكن للمهاجم الانتقال من الاختبار العام إلى استراتيجيات الاستغلال (Exploit) المستهدفة بدقة.

| **حالة الاختبار** | لم يتم التنفيذ |

### هل تكشف ترويسات استجابة HTTP عن اسم إطار العمل أو إصداره؟
- [ ] لا — الترويسات مثل `X-Powered-By` أو `Server` غير موجودة أو تم تنظيفها (Sanitized)  
- [ ] نعم — اسم إطار العمل موجود ولكن الإصدار **مخفي**  
- [ ] نعم — اسم إطار العمل والإصدار **كلاهما مكشوفان**  

### هل يمكن تحديد ملفات تعريف الارتباط أو هياكل الأدلة الخاصة بإطار العمل؟
- [ ] لا — العناصر الشائعة لإطار العمل (مثل مسارات `.js` أو أسماء ملفات تعريف الارتباط) مموهة (Obfuscated) أو تم تغيير اسمها  
- [ ] نعم — ملفات تعريف الارتباط الخاصة بإطار العمل (مثل `JSESSIONID` أو `PHPSESSID` أو `csrftoken`) **قابلة** للتحديد  
- [ ] نعم — هياكل الأدلة (مثل `/wp-content/` أو `_next/static/`) **مرئية** وتؤكد نوع إطار العمل  

### هل تفصح رسائل الخطأ أو تعليقات الكود المصدري عن تفاصيل إطار العمل؟
- [ ] لا — معالجة الأخطاء عامة وتمت إزالة التعليقات  
- [ ] نعم — آثار تتبع الأخطاء (Stack traces) الخاصة بإطار العمل **مُفعلة** ومرئية في الاستجابات  
- [ ] نعم — يحتوي الكود المصدري HTML على تعليقات أو وسوم بيانات وصفية (Metadata tags) تحدد إطار العمل  

### هل توجد ثغرات معروفة مرتبطة بإصدار إطار العمل الذي تم تحديده؟
- [ ] لا — إطار العمل المحدد محدث مع **عدم وجود** ثغرات عامة معروفة  
- [ ] نعم — إصدار إطار العمل قديم ويمكن تحديد ثغرات (CVEs) معينة
