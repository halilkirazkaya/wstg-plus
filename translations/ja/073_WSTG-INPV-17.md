## WSTG-INPV-17 — ホストヘッダーインジェクションのテスト (Testing for Host Header Injection)

ホストヘッダーインジェクション (Host Header Injection) は、アプリケーションがユーザーから提供された `Host` ヘッダーを信頼し、十分な検証を行わずにリンク生成やリダイレクトなどのサーバーサイドのロジックに組み込んだ場合に発生します。攻撃者はこのヘッダーを操作することで、Webキャッシュ汚染 (Web Cache Poisoning) を引き起こしたり、機密性の高いトークンを外部ドメインにリダイレクトさせることでパスワードリセット汚染 (Password Reset Poisoning) を助長したり、ヘッダーベースのルーティングに依存するセキュリティ制御をバイパスしたりします。攻撃に成功すると、攻撃者はセッションデータを流出させたり、アカウント乗っ取り (Account Takeover) を実行したり、疑いのないユーザーを悪意のあるインフラストラクチャにリダイレクトさせたりすることが可能になります。この脆弱性は、ロードバランシング環境や、リクエストのコンテキストに基づいて絶対URLを動的に生成するアプリケーションで非常によく見られます。


| フィールド | 値 |
|---|---|
| **WSTG ID** | WSTG-INPV-17 |
| **CWE** | CWE-20 |
| **テストステータス** | 実施なし (Not Performed) |
| **深刻度** | 中 (Medium) / 高 (High)* |

> *パスワードリセットなどの電子メール内で機密性の高いリンクを生成するために `Host` ヘッダーが使用されている場合、深刻度は「高」になります。

### アプリケーションは、リンク、リダイレクト、またはスクリプト内で `Host` ヘッダーの値を反映していますか？
- [ ] いいえ — アプリケーションはハードコードされたベースURL、または厳格に検証された設定を使用している
- [ ] はい — `Host` ヘッダーは反映されているが、ホワイトリストに対して厳格に検証されている
- [ ] はい — `Host` ヘッダーは反映されており、形式不備の値 (例: `expected.com:attacker.com`) によるバイパスが可能である
- [ ] はい — `Host` ヘッダーが検証なしで反映されている

### 代替のホスト関連ヘッダーがアプリケーションまたはインフラストラクチャによって処理されていますか？
- [ ] いいえ — `X-Forwarded-Host`、`X-Host`、`Forwarded` などのヘッダーは無視される
- [ ] はい — 代替ヘッダーは処理されるが、内部ロジックを上書きすることはない
- [ ] はい — 代替ヘッダーを使用して、プライマリの `Host` ヘッダーの値を上書きできる

### `Host` ヘッダーを操作して、システムが生成する電子メール（パスワードリセットなど）を汚染できますか？
- [ ] いいえ — 電子メールのリンクは、サーバー設定にある静的で信頼されたドメインを使用している
- [ ] はい — `Host` ヘッダーは電子メールのリンクに影響を与えるが、検証をバイパスすることはできない
- [ ] はい — `Host` ヘッダーを操作して、パスワードリセットトークンを攻撃者が制御するドメインにリダイレクトさせることができる *(緊急/Critical)*

### アプリケーションは `Host` ヘッダーによる Webキャッシュ汚染 (Web Cache Poisoning) の影響を受けますか？
- [ ] いいえ — キャッシュメカニズムが存在しない、または `Host` ヘッダーがキャッシュキーの一部である
- [ ] はい — キャッシュは存在するが、キーに含まれない入力に対して `Host` ヘッダーを厳格に検証または無視している
- [ ] はい — キャッシュが存在し、注入された `Host` ヘッダーによって汚染され、他のユーザーに悪意のあるコンテンツを提供する可能性がある

### サーバーはバーチャルホストのルーティングにおいて、任意の `Host` ヘッダーを受け入れますか？
- [ ] いいえ — 認識されない `Host` ヘッダーに対して、サーバーはエラーまたはデフォルトページを返す
- [ ] はい — サーバーは任意の `Host` ヘッダーを受け入れる。これは偵察 (Reconnaissance) や内部サービス列挙 (Internal service enumeration) に有用である
