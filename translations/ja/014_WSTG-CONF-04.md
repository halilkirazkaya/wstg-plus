## WSTG-CONF-04 — 機密情報に関する古いバックアップファイルおよび参照されていないファイルの確認

古いバックアップファイル (Backup) および参照されていないファイルの確認には、一般公開を意図していない、ウェブサーバー上の忘れられたファイル、一時ファイル、または隠しファイルを特定することが含まれます。これらのファイルには、ソースコード (Source Code) のバックアップ (`.zip`, `.bak`)、テキストエディタのスワップファイル (Swap Files) (`.swp`, `~`)、または機密情報を漏洩させる可能性のあるソース管理 (Source Control) メタデータ (`.git`, `.svn`) が含まれることがよくあります。攻撃者 (Attackers) は、自動化されたワードリスト (Wordlists) や探索ツール (Discovery Tools) を使用して、一般的な命名規則をブルートフォース (Brute-force) し、データベースの資格情報 (Credentials)、ハードコードされた API キー (API Keys)、またはさらなるエクスプロイト (Exploit) を容易にするロジックを窃取 (Exfiltrate) することを目的としています。この脆弱性は通常、手動でのサーバーメンテナンスや、本番環境のウェブルートをクリーンアップできない不備のある CI/CD パイプライン (CI/CD pipelines) に起因します。

| **CWE** | CWE-530 |
| **テストステータス** | 未実施 |

> *設定ファイル、ソースコードのアーカイブ、または資格情報が発見された場合、深刻度は「高」となります。

### ウェブサーバーでディレクトリリスティング (Directory Listing) が有効になっていますか？
- [ ] いいえ — ディレクトリリスティングは**無効化**されており、403 Forbidden またはカスタムエラーを返します
- [ ] はい — 機密性のないディレクトリでディレクトリリスティングが**有効**になっています
- [ ] はい — ソースコードや機密ファイルを含むディレクトリでディレクトリリスティングが**有効**になっています *(致命的)*

### 一般的な拡張子（例：.bak, .old, .save）を持つバックアップファイルは発見可能ですか？
- [ ] いいえ — 一般的なバックアップ拡張子は**見つからない**か、サーバー設定によってブロックされています
- [ ] はい — バックアップファイルは存在しますが、機密情報は**含まれていません**
- [ ] はい — 設定ファイルまたはソースコードのバックアップファイルにアクセス**可能**です *(高)*

### ソース管理ディレクトリ（例：.git, .svn）またはメタデータが露出していますか？
- [ ] いいえ — ソース管理メタデータは**存在しない**か、正しくブロックされています
- [ ] はい — メタデータは存在しますが、リポジトリ全体の再構築は**できません**
- [ ] はい — 露出したメタデータを介してソースコードリポジトリ全体を**窃取可能**です

### アプリケーションの圧縮アーカイブ（例：.zip, .tar.gz, .rar）がウェブルートに存在しますか？
- [ ] いいえ — ブルートフォースや列挙によって機密性の高いアーカイブファイルは**発見されませんでした**
- [ ] はい — アーカイブは見つかりましたが、パスワードで保護されているか、公開アセットのみが含まれています
- [ ] はい — アプリケーションのソースまたはデータを含む暗号化されていないアーカイブが公開**アクセス可能**です

### アプリケーションはテキストエディタや IDE によって作成された一時ファイルを漏洩させていますか？
- [ ] いいえ — `.swp`, `~`, `.DS_Store` のような一時ファイルは**存在しません**
- [ ] はい — 機密性のない一時ファイルが**存在します**
- [ ] はい — 一時ファイルにより、ソースコードのスニペットや内部パスが**判明します**
