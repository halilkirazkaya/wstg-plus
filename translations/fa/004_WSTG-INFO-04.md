## WSTG-INFO-04 — شناسایی و شمارش اپلیکیشن‌ها در وب‌سرور

شمارش اپلیکیشن‌ها (Application Enumeration) فرآیند شناسایی تمامی اپلیکیشن‌های وب و سرویس‌هایی است که روی یک وب‌سرور یا آدرس IP واحد میزبانی می‌شوند. از آنجا که وب‌سرورهای مدرن اغلب چندین میزبان مجازی (Virtual Hosts)، محیط‌های آزمایشی قدیمی (Legacy staging environments) یا رابط‌های مدیریتی (Administrative interfaces) را روی پورت‌ها و مسیرهای غیراستاندارد میزبانی می‌کنند، عدم شناسایی این دارایی‌های پنهان باعث می‌شود بخش قابل توجهی از سطح حمله (Attack Surface) تست‌نشده باقی بماند. مهاجمان از تکنیک‌هایی مانند انتقال ناحیه DNS (DNS zone transfers)، حملات جستجوی فراگیر میزبان مجازی (Virtual host brute-forcing) و جستجوی معکوس IP (Reverse IP lookups) برای کشف این نقاط ورود فراموش‌شده یا مبهم استفاده می‌کنند که معمولاً امنیت کمتری نسبت به اپلیکیشن اصلی عملیاتی دارند.

### آیا چندین آدرس IP یا نام‌های مستعار DNS با زیرساخت هدف مرتبط هستند؟
- [ ] خیر — تنها یک IP واحد و رکورد A وجود دارد  
- [ ] بله — چندین آدرس IP یا نام مستعار شناسایی شده‌اند که دامنه ارزیابی را گسترش می‌دهند  

### آیا وب‌سرور چندین میزبان مجازی (vhosts) را روی یک IP واحد میزبانی می‌کند؟
- [ ] خیر — تنها دامنه اصلی توسط سرور ارائه می‌شود  
- [ ] بله — میزبان‌های مجازی اضافی شناسایی شده‌اند اما دسترسی به آن‌ها امکان‌پذیر نیست (مثلاً با خطای 403 Forbidden)  
- [ ] بله — میزبان‌های مجازی مخفی، توسعه یا آزمایشی شناسایی شده‌اند و قابل دسترسی هستند  

### آیا اپلیکیشن‌ها روی پورت‌های غیراستاندارد میزبانی می‌شوند؟
- [ ] خیر — تنها پورت‌های استاندارد (80/443) باز هستند  
- [ ] بله — پورت‌های غیراستاندارد باز هستند اما اپلیکیشن وب میزبانی نمی‌کنند  
- [ ] بله — اپلیکیشن‌های وب اضافی روی پورت‌های غیراستاندارد (مانند 8080، 8443، 9000) شناسایی شده‌اند  

### آیا اپلیکیشن‌های مجزا به مسیرهای URI خاصی نگاشت شده‌اند؟
- [ ] خیر — یک اپلیکیشن واحد کل دایرکتوری ریشه (Root) را مدیریت می‌کند  
- [ ] بله — چندین اپلیکیشن از طریق شناسایی دایرکتوری (Directory enumeration) کشف شده‌اند (مانند `/api`, `/portal`, `/backup`)  

### آیا سرویس‌های شناسایی شخص ثالث (Third-party reconnaissance) اپلیکیشن‌های قدیمی یا ثانویه را نشان می‌دهند؟
- [ ] خیر — یافته قابل توجهی از `Shodan` ، `Censys` یا `Wayback Machine` به دست نیامده است  
- [ ] بله — تصاویر ثبت‌شده قدیمی یا اسکن‌های سرویس، اپلیکیشن‌هایی را نشان می‌دهند که در حال حاضر فعال هستند اما لینک‌دهی نشده‌اند
